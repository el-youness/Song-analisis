# Song-analisis - NLP
The goal of this project is to use the R language and use multiple techiques like Regression, Clustering ...etc.<br/>
Buisness Goal: Artists with lack of inspiration can use our product to generate song lyrics of a certain artist and might just find the inspiration they need.

## First delivery (first delivery.ipynb)
- Cleaning the dataset: remove pronouns and determiners from lyrics ...
- Statistical analysis of the dataset: wordcount ...
- Sentiment analysis: group the songs by sentiment using the words
- Curse words analysis: top 50 songs with curse words in them are rap songs
